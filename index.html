Software Engineering Intern @ Alarm.com

In summer 2023, I worked at Alarm.com, where I worked on a full-stack project involving Ember.js, C#, Microsoft SQL Server, and ASP.NET

My project was creating a customer feedback parsing tool, as well as the feedback page itself. I created the page in Ember.js, then used C#/.NET to connect to an LLM at certain intervals and send feedback to be parsed. I also had to write appropriate SQL scripts to automate this parsing and sending process, along with the appropriate OWASP security practices. Finally, I created the internal page using ASPX to surface the condensed insights, which will improve the company's review and response process.

Through this internship, I really got a taste of what it's like to work as a software engineer, getting experience with the non-technical parts as well, including using JIRA, daily stand-ups, communicating with PMs and QEs regarding tickets, and the work environment in general.


Programming Intern @ KORE Wireless

In summer 2022, I worked remotely for the Atlanta-based company KORE Wireless, where I got to become familiar with IoT and embedded systems.

My role with KORE was to create automated test scripts to monitor IoT devices' functionality. I did this primarily in micropython, but for many of these tests, I had to create an endpoint in Node-RED to ping or receive a response from.

Through this internship, I got a feel for working with embedded systems: in my case, a BG-95 modem which was connected to an ESP-32 microcontroller. I learned a lot about IoT, and being my first tech internship, I got familiar with many industry tools, both technical and non-technical.
