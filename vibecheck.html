Vibe Check

This was a final project for my Computer Vision class at Brown. After mulling over many ideas, our group decided to create a TikTok style filter generator. We used a vision-based model in order to try to classify a song into one of ten genres. Based on the genre, we would look for genre-specific features. For example, hip-hop music would look for bass hits, while reggae music would look for drum hits.

After we had the genre and the genre-specific features, it was time to create a different filter for each genre. For disco music, for example, we used a simple color filter that changed color to the beat of the song, which resembled a classic disco. For rock music, we added a global noise filter along with a greyscale filter to add a vintage look, and added a sobel kernel during loud moments to give the appearance of the lights going out at a rock concert. This filter was applied to a user inputted video in order to create a TikTok style filter. Finally, we created GUI where users could record a video, add music, and the filter would automatically applied.

All in all, we created a highly accurate model. It achieved 88 percent genre-classification accuracy on two-second clips of music. In order to make the model even higher accuracy, we sampled several 2-second clips from an inputted song, and took a majority vote on what genre the song was. This led to near 100 percent accuracy in practice. The music feature extraction and filter generation was a fun way to apply the model as part of a larger project.

Please enjoy these result videos. Note that effects and effect timing were both determined by genre classification: disco and classical, respectively. These videos have sound.
