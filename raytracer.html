Realtime Raytracer

This realtime raytracer accepts a scenefile, and beautifully illuminates the scene via the phong illumination model.

As this is a 3D renderer, you may include objects at different depths, different kinds of light (point light, spot light, and directional light), and it creates a 3D rendering. The lighting scheme includes reflections via recursive raytracing, and the color of each object is determined by the recursive phong illumination model, which takes into account ambient lighting, the diffuse component, and the specular, or shiny component. It also supports texture mapping for objects that aren't solid colored.

This was one of the hardest and most challenging time consuming projects I completed in college. As it's an assignment that will be re-used, a GitHub link is unavailable, but feel free to ask me any questions about this project. Please enjoy some of the coolest scenes I was able to render.
