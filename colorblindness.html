Colorblindness Emulator

This project was initially the result of me coming across Agathe Badia's talk at EmberConf where she presents this extension; I simply implemented it myself. However, I learned through this process how to create a Chrome extension, while also honing my JavaScript skills and learning more about accessibility. In fact, the reason I came across this article in the first place was because I wanted to learn more about accessibility in web design.

I am passionate about creating accessible websites, and I came across this talk while learning about A11y standards, which I came across in my last internship at Alarm.com. I found that it was difficult to understand a lot about why contrasting colors could be difficult to see as somebody who is not colorblind, so I implemented this extension and tested it on a few websites. I found that websites that were A11y compliant, or that scored high on the A11y or WAVE validators were pretty easy to see even when applying all forms of colorblindness, while those that weren't, I had trouble when emulating certain types of colorblindness.

All in all, this was a simple project that I learned a lot about accessibility standards from. I also got to hone my JavaScript and HTML too, while learning about Chrome extensions.
