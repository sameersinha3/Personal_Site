Application Manager

I first created this project to get familiar with jQuery and PHP due to an internship I applied to requiring familiarity with how to interact with SQL databases using these technogies. After watching some online tutorials, I figured it was best to get my hands dirty and figure it out myself. This project required a steep learning curve, as I had never used PHP or jQuery before, although I had some familiarity with SQL.

This application is meant to be an application manager. One context this app could be use would be food trucks applying to be food vendors at a large music festival. Each applicant would be able to submit an application, while an administrator of the event could create applications manually, as well as read, edit and delete any applications submitted by the vendors.

Technically, this really only involves a table of the vendors in SQL, but we use PHP and jQuery to interact with this table in different ways. Below is an image of the administrator view, with read, edit and delete actions. Note that the create button is elsewhere in the app.
